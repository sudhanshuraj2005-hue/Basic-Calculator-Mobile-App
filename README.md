# 🧮 Basic Calculator - Android App

This is a simple and user-friendly calculator application developed for Android.  
The app performs basic arithmetic operations and demonstrates core Android development concepts such as layouts, event handling, and activity lifecycle.

## 🚀 Features
- Perform basic operations: Addition, Subtraction, Multiplication, Division
- Clean and responsive user interface
- Real-time input and result display
- Lightweight and easy to use

## 🛠️ Built With
- **Language:** Java
- **IDE:** Android Studio
- **UI Design:** XML Layouts
